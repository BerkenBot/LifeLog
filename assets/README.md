# Assets

Place your app assets in this folder:

## Required Files

### icon.png
- **Size:** 1024 x 1024 pixels
- **Format:** PNG (no transparency for iOS)
- **Used for:** App icon on home screen and App Store

### splash.png  
- **Size:** 1284 x 2778 pixels (or similar tall aspect ratio)
- **Format:** PNG
- **Used for:** Launch screen while app loads

## Creating Icons

You can use tools like:
- [Figma](https://figma.com) - Free design tool
- [Canva](https://canva.com) - Easy icon creation
- [SF Symbols](https://developer.apple.com/sf-symbols/) - Apple's icon library

## Generating All Sizes

Once you have a 1024x1024 icon, Expo/EAS will automatically generate all required sizes during build.

## Suggested Icon Design

For a check-in/wellness app, consider:
- Simple checkmark or heart shape
- Soft, calming colors (blues, greens)
- Rounded corners (iOS will mask it anyway)
- High contrast for visibility at small sizes
